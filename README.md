## インストール / Installation
1. リポジトリをダウンロード / clone repository
2. `a1111_webui_pth.json`にて`AUTOMATIC1111/Stable-Diffusion-WebUI`の位置を設定 / edit `a1111_webui_pth.json` for your opt
3. `v5β.bat`を実行 / launch `v5β.bat`
